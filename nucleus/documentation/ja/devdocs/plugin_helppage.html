ヘルプファイルの種類

上から順に最初に見つかったものが適用されます。
形式配置場所
php プラグインのフォルダ/help.php
htmlプラグインのフォルダ/help.html
phpプラグインのフォルダ/help/index.php
markdownプラグインのフォルダ/help/help-[ja|en].md
[]は 現在のユーザーのローケル
markdownプラグインのフォルダ/help/help.md
markdownプラグインのフォルダ/help/help-en.md
htmlプラグインのフォルダ/help/index.html

ヘルプファイル - markdown

ファイル名

言語を指定しない場合は、プラグイン名/help/help.md
日本語の場合は、プラグイン名/help/help-ja.md


プラグインwikiからの取り込み


Nucleus CMS 日本 プラグイン: http://japan.nucleuscms.org/wiki/plugins

html → markdown

$ cat help.html | pandoc -f html -t markdown_github

dokuwiki → markdown

$ curl http://japan.nucleuscms.org/wiki/plugins:名称\?do\=export_xhtmlbody | pandoc -f html -t markdown_github

例: skinfiles
$ curl http://japan.nucleuscms.org/wiki/plugins:skinfiles\?do\=export_xhtmlbody | pandoc -f html -t markdown_github

markdown → dokuwiki

$ cat filename.md | pandoc -f markdown_github -t dokuwiki

pandoc

Debian/Ubuntu系のOSで利用できます。
Windows: WSL2(Ubuntu)を利用してください。Cygwin用のパッケージはないようです。

$ sudo apt install pandoc
