更新履歴

Nucleusマニュアルに戻る
  • Nucleus 3.80 (3.8.0) (未定)
    • [変更] NucleusCMS本体のサポートされるPHPのバージョンが変更されました。プラグインも使用するPHPのバージョンに対応している必要あります。
    • [変更] データベース：PDOドライバーへ完全移行しました。
    • [新規] 新規インストールの際にインストール認証の設定が必要になりました。
    • [新規] 「プラグイン更新通知」「プラグインダウンロード」が追加されました。
    • [新規] 「アイテムの表示期限」が追加されました。
    • [廃止] EUC-JPを含む文字コードの扱いを終了しました。
    • [廃止] mysql mysqliドライバーでの接続サポートを終了しました。
    • [新規] データベースの最適化/修復が追加されました。
    • [新規] メンバーの停止(アカウントロック)が追加されました。
      有効にするとメンバーはログインやパスワード発行などの各種操作ができなくなります。
    • [新規] Blogオプション authorvisibleが追加されました。
    • [新規] 投稿画面に現在時刻を挿入するボタンが追加されました
    • [新規] データベース MySQL utf8mb4への対応。
    • [新規] コンテンツの圧縮をサポート(gzip, deflate)
    • [新規] アイテム編集後の移動先を選べるようになりました。
    • [新規] Skinクラスに関数が追加されました(changeSkinByName, changeSkinById, getRootURL, getURL)。
    • [新規] グローバル変数 $DB_DRIVER_NAME が追加されました。現在設定できる値は mysql sqlite です
    • [廃止] 次のsupportsFeature が廃止されました [SqlTablePrefix, SqlApi](sql_table関数の十分な周知期間が経過したため)。プラグインのロード時に これらのsupportsFeature のチェックは行われなくなりました。
    • [変更] 次の列の長さが(20から50)に変更されました : nucleus_plugin_option.oname, nucleus_config.name
    • [変更] インストーラー：メールアドレスに ～@localhost を使用できるように変更しました。
    • [変更] スキンテーブルに 列spartstypeを追加しパーツとページを分離しました。
    • [変更] プラグインロード中のエラー発生時に実行が中断されないようにtryエラー処理を追加しました。
    • [変更] $CONF['DisableRSS'] が追加されました。グローバル設定: RSSの出力を有効にする ( xml-rss2.php , rsd.php , atom.php )。出力をしない場合は、関連ファイルの更新が必要です。
    • [廃止] nucleus/forgotpassword.htmlが削除されました。
    • [変更] スキン/テンプレート変数に <%NP_PluginName(parameters)%> が追加されました。
    • [変更] プラグインクラスのgetEventList関数が不要になりました。プラグインのevent_*関数は親クラスの_getEventList関数により自動的に取得されます。
